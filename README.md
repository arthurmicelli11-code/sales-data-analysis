# sales-data-analysis

## Description
Beginner data analysis project focused on exploring sales data using Python and SQL.

## Objectives
- Analyze overall sales performance
- Identify top-selling products
- Understand sales trends over time
- Compare sales by region

## Planned Steps
1. Collect a sample sales dataset
2. Clean and organize the data using Python (Pandas)
3. Perform basic aggregations and calculations
4. Use SQL queries to analyze sales patterns
5. Create simple tables and charts using Google Sheets

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL
- Google Sheets
- Statistics
